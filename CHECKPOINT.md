### Как да провериш локално

```bash
npm install
npx prisma db push
npm run db:seed
npm run dev
```

- Магазин: http://localhost:3000  
- Админ: http://localhost:3000/admin (след вход)  
- Демо плащане: добави в количка → Плащане → „Демо плащане“
